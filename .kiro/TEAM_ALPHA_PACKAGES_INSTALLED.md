## ğŸ’¡ Installation Notes

### Why --legacy-peer-deps?
The project uses React 19, but some dependencies (like `vaul`) haven't updated their peer dependencies yet. Using `--legacy-peer-deps` allows installation while maintaining compatibility.

### Is This Safe?
Yes! The packages work fine with React 19. The `--legacy-peer-deps` flag just bypasses the peer dependency check.

### Future Updates
When dependencies update to support React 19, you can reinstall without the flag:
```bash
npm install @fullcalendar/react --save
```
