## ðŸ”§ Technical Stack

### Frontend (Hospital Management System)
- **Framework**: Next.js 16 + React 19
- **UI Library**: Radix UI + Tailwind CSS 4.x
- **Forms**: React Hook Form + Zod
- **State**: React Query for server state
- **Charts**: Recharts for analytics
- **Real-time**: WebSocket for live updates

### Mobile (React Native)
- **Framework**: React Native 0.73+
- **Navigation**: React Navigation 6+
- **State**: Zustand for local state
- **Offline**: WatermelonDB for local storage
- **Push**: Firebase Cloud Messaging
- **Auth**: AWS Cognito integration

### Backend Additions
- **Search**: PostgreSQL Full-Text Search
- **Notifications**: AWS SES (email) + Twilio (SMS)
- **Real-time**: WebSocket server
- **Reporting**: PDF generation with Puppeteer
- **Export**: CSV/Excel generation
